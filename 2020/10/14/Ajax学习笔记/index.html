Ajax 传送数据的3种方式

(1 )xml：笨重，解析困难，但是 xml 是通用的数据交换模式。

(2) html：不需要解析直接放到文档中，若尽更新一部分数据，但传送数据不是很方便，且HTML代码需要拼装完成。

(3) JSON：小巧，有面向对象的特性，且第三方的 jar 包可以把 java 对象或集合转化为 JSON 字符串（用的最多）
